Este es el codigo fuente para la API utilizada en "https://agro-costa.com/" para cotizar precios utilizando el número de referencia de los distintos articulos disponibles en el inventario. Las principales funcionalidades de este codigo son:
  1. Permite tanto a usuarios registrados como no registrados acceder a la base de datos de la compañía para hacer consultas sobre los articulos disponibles.
  2. Discrima quién tiene derecho a ver qué dependiendo de si son o no usuarios registrados y que permisos tienen en el sistema.
  3. Permite al ADMIN el manejo de permisos individuales para cada usuario.
  4. Permite al ADMIN la creación de usuarios en la base de datos.
  5. Permite al ADMIN el manejo de las distintas lineas presentes para los productos.
